# ExOCRists

## How to run:

- minicli for tests:
    - `make minicli`
    - `./minicli`
- xor:
    - `make xor`
    - `./neuralNetwork/xor`
- image treatment example:
    - `make imagetreatment`
    - `./imageTreatment/main`
- all:
    - `make all`
    - now executables are in the `bin/` folder
        - `./bin/xor`  for the xor
        - `./bin/main` for the image treatment
        - `./bin/minicli` for everything
